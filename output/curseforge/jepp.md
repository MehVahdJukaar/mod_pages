<p style="text-align:center"><span style="font-size:24px"><strong>📖 About 📖</strong></span></p>

<p><span style="font-size:18px">JEPP is a JEI and REI addon that adds a recipe category for paintings, so you can look up which paintings exist instead of placing one repeatedly until you get the one you wanted.</span></p>

<p style="text-align:center"><img src="https://raw.githubusercontent.com/MehVahdJukaar/mod_pages/master/assets/separators/jepp.png" alt="separator" width="1688" height="42"></p>

<p style="text-align:center"><span style="font-size:24px"><strong>🖼️ Features 🖼️</strong></span></p>

<ul>
<li><span style="font-size:18px">Press <strong>U</strong> on a painting to open a window listing every painting available</span></li>
<li><span style="font-size:18px">Each entry shows the painting's name, its dimensions and which mod adds it</span></li>
<li><span style="font-size:18px">Works with any mod that adds paintings, with no configuration</span></li>
</ul>

<p style="text-align:center"><img src="https://raw.githubusercontent.com/MehVahdJukaar/mod_pages/master/assets/separators/jepp.png" alt="separator" width="1688" height="42"></p>

<p style="text-align:center"><span style="font-size:24px"><strong>❓ F.A.Q. ❓</strong></span></p>

<p><span style="font-size:18px"><strong>Q: Do I need JEI or REI?</strong></span></p>

<p><span style="font-size:18px"><strong>A:</strong> One of the two, yes. JEPP is a plugin for them, it has no UI of its own.</span></p>

<p><span style="font-size:18px"><strong>Q: Is it client-side?</strong></span></p>

<p><span style="font-size:18px"><strong>A:</strong> Yes, it only adds a lookup screen. No server install needed.</span></p>

<p><span style="font-size:18px"><strong>Q: Does it work with paintings from other mods?</strong></span></p>

<p><span style="font-size:18px"><strong>A:</strong> Yes, automatically. Anything registered as a painting shows up.</span></p>
