<p style="text-align:center"><span style="font-size:24px"><strong>📖 About 📖</strong></span></p>

<p><span style="font-size:18px">Not just a performance mod.</span></p>

<p><span style="font-size:18px">Fast Paintings replaces painting entities with their block equivalent — baked models instead of an entity renderer. This makes them as fast as blocks can be, and unlocks a bunch of side benefits.</span></p>

<p style="text-align:center"><img src="https://raw.githubusercontent.com/MehVahdJukaar/mod_pages/master/assets/separators/blue.png" alt="separator" width="1688" height="42"></p>

<p style="text-align:center"><span style="font-size:24px"><strong>⚡ Performance ⚡</strong></span></p>

<ul>
<li><span style="font-size:18px">Zero painting rendering lag — baked block models are as cheap as any other block</span></li>
<li><span style="font-size:18px">No entity tick overhead — they're blocks now, not entities</span></li>
<li><span style="font-size:18px">Paintings no longer disappear when you're far away</span></li>
<li><span style="font-size:18px">Better visual quality via mipmap support</span></li>
<li><span style="font-size:18px">Paintings react immediately when their supporting block is broken</span></li>
<li><span style="font-size:18px">Won't accidentally be killed by the <code>/kill</code> command</span></li>
</ul>

<p style="text-align:center"><img src="https://raw.githubusercontent.com/MehVahdJukaar/mod_pages/master/assets/separators/blue.png" alt="separator" width="1688" height="42"></p>

<p style="text-align:center"><span style="font-size:24px"><strong>🖼️ Customization 🖼️</strong></span></p>

<p><span style="font-size:18px">Because paintings are now blocks with JSON models, you can override their appearance entirely with a resource pack.</span></p>

<p><span style="font-size:18px">Want wooden frames? Painted borders? Rounded corners? Edit the model JSON for each painting part (<code>top_left</code>, <code>top_right</code>, etc.) and apply any geometry you like.</span></p>
