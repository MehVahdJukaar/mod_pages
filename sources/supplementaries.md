## 📖 About 📖

Supplementaries is a large, highly configurable Vanilla+ mod focused on adding functional and decorative content that feels at home in the base game. Every feature is designed to be aesthetically viable and complementary to existing content, filling gaps that Mojang left behind.

The mod adds over 80 blocks and items across many categories: rope, jars, sign posts, pedestals, sconces, notice boards, wind chimes, globes, plushies, bamboo spikes, turnstiles, and much more. Check the [**Wiki**](https://github.com/MehVahdJukaar/Supplementaries/wiki) for the full feature list.

- Requires **[DEPENDENCY]**
- While supported, the Fabric version may be missing some minor features
- Free to include in modpacks

## ❓ F.A.Q. ❓

**Q: How do I disable or tweak a feature?**
**A:** Refer to the Customization section in the [**Wiki**](https://github.com/MehVahdJukaar/Supplementaries/wiki) or join our [**Discord**](https://discord.com/invite/qdKRTDf8Cv).

**Q: Could you update/downgrade to version X?**
**A:** I plan to port the mod to every new major Minecraft version eventually, be patient. Downgrades are not happening. Maintaining a single version already takes an enormous amount of time, and so much changed between versions that it would mean re-learning modding from scratch.

**Q: Could you add feature/block X?**
**A:** Join our [Discord](https://discord.com/invite/qdKRTDf8Cv), we have a dedicated suggestions section. If the idea fits the mod well it has a good chance of eventually being added.

**Q: How do I use block/item X?**
**A:** Refer to the [**Wiki**](https://github.com/MehVahdJukaar/Supplementaries/wiki).

**Q: The Fabric version is missing feature X**
**A:** Supplementaries is primarily a Forge mod. Missing Fabric features didn't have a straightforward implementation because Fabric lacked a counterpart to something the Forge API provides. For the full picture, see [this](https://i.imgur.com/JdipbyU.jpg).
