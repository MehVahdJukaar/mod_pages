## 📖 About 📖

Labels are like Item Frames - except they work on every surface and shape, sit flush against the block, and let you interact with whatever is behind them.

Place a Label on any surface, then shift-click it with an item to set what it displays. The displayed icon is rendered in a pixelated, colored style that looks deliberately hand-made.

[SEPARATOR]

## 🏷️ Features 🏷️

- **Universal placement** - sticks to floors, ceilings, walls, slabs, stairs, and fences; no flat wall required
- **Transparent interaction** - clicking a Label passes through to the block behind it, so you can open a chest, pull a lever, or access a barrel without removing the label
- **Glow Ink Sac** - right-click a placed Label with Glow Ink to make it visible in the dark
- **Dyeable** - right-click with any dye to tint the label's background color; 16 colors available
- **Feather** - shift-click with a Feather to toggle display of the held item's name instead of its icon
- **Item stacks** - clicking with a stack updates the display item; the Label itself retains no inventory
- Fully configurable rendering style via configs and resource packs

[SEPARATOR]

## ❓ F.A.Q. ❓

**Q: Can I put a Label on a chest without blocking access?**
**A:** Yes - clicking the Label interacts with the block behind it, so your chest opens normally.

**Q: Does it work on modded blocks?**
**A:** Labels can be placed on any surface that accepts a block entity interaction. Works with most modded storage blocks.
