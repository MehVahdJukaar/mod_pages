<span style="font-size: 18px;"><strong>A:</strong>&nbsp;Refer to the Customization section in the <a href="https://github.com/MehVahdJukaar/Supplementaries/wiki"><strong>Wiki</strong> </a>or join our <a href="https://discord.com/invite/qdKRTDf8Cv"><strong>Discord</strong></a></span>

<span style="font-size: 18px;"><strong>Q: Could you update/downgrade to X version?</strong></span>
<span style="font-size: 18px;"><strong>A:</strong>&nbsp;I plan to eventually port the mod to every new major Minecraft version so be patient.<br>As for downgrades absolutely no, I'm not interested in supporting older obsolete versions. Maintaining this one version already takes up so much of my time and so much changed between those versions that I would have to pretty much re-learn how to mod.</span>

<span style="font-size: 18px;"><strong>Q: Could you add X feature/block?</strong></span>
<span style="font-size: 18px;"><strong>A:</strong> I would encourage you to join our Discord. We have a section dedicated to suggestions like that. If I'll feel like your idea fits well enough it will have a high chance to be eventually added.</span>

<span style="font-size: 18px;"><strong>Q: How do I use X block/item?</strong></span>
<span style="font-size: 18px;"><strong>A:</strong> Once again refer to the <strong>Wiki.</strong></span>

<span style="font-size: 18px;"><strong>Q: How can I fix this issue/ crash?</strong></span>
<span style="font-size: 18px;"><strong>A:</strong>&nbsp;Please open a report in the Issues section always including a <strong>Crash Report.</strong></span>

<span style="font-size: 18px;"><strong>Q: Fabric version is missing X</strong></span>
<span style="font-size: 18px;"><strong>A:&nbsp;</strong>Supplementaries is a mainly a Forge Mod and if some feature is missing that means that it didnt have a simple or easy implementation as Fabric was missing a counterpart of something provided by Forge API that such feature was heavily reliant on.
